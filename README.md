### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/tictactoe.git
   cd tictactoe
   ```

2. **Install dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to `http://localhost:5173`
